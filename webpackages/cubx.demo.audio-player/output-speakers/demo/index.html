IDMT output speakers

This component receives audio samples via the samples input slot and plays it back. The processing is in realtime so enough audio data for interruption-free sound should be provided. The input format is Float32Array with a blocksize of 2048 samples, values from -1 to 1 and a sample rate of 44100 Hz. If more samples are handed over then needed the data is buffered.

Volume can be set with values between 100 (loud) and 0 (silent).

The example uses an Oscillator for generating sound samples and passes hands them over to the component for playback.
